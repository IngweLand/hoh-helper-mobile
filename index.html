HoH Helper Mobile

HoH Helper Mobile allows you to export your Heroes of History game data—including city layouts, heroes, equipment, leaderboards, and more—to Forge of Games. It is designed to help players gain deeper insights into their game progress and performance.

This version is specifically intended for players who primarily or exclusively play the game on mobile devices. If you're using a desktop browser, we recommend our browser extension for a more streamlined experience.

This is a third-party tool and is not affiliated with or supported by InnoGames.

To function properly, the tool requires your real in-game username and password.

If you are a first-time user, please read the help section to understand how this tool works and to review the security implications.
